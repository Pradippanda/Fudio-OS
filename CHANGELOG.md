Versioning

Fudio OS follows Semantic Versioning (SemVer):

- MAJOR version: Breaking changes.
- MINOR version: New features that remain backward compatible.
- PATCH version: Bug fixes and small improvements.

Examples

- "1.0.0" → First stable release
- "1.1.0" → New features
- "1.1.1" → Bug fixes
- "2.0.0" → Breaking changes
