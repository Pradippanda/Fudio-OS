# Fudio OS

Fudio OS is an AI-powered Restaurant SaaS platform built to help cafes and restaurants manage their business from one dashboard.

## Features
- 🍽️ Menu Management
- 📦 Order Management
- 📊 Analytics Dashboard
- 👥 Customer CRM
- 🤖 AI Assistant
- 📱 Mobile-first Design
- 🌐 Multi-branch Support
