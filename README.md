🚀 Overview

Fudio OS is an open-source restaurant operating system that helps restaurants manage their daily operations from a single dashboard.

Whether you run a café, cloud kitchen, food truck, or multi-branch restaurant, Fudio OS aims to simplify operations with modern tools and AI-powered workflows.

⚙ Installation

Clone the repository

git clone https://github.com/Pradippanda/Fudio-OS.git

Move into the project

cd Fudio-OS

Install dependencies

npm install

Start development server

npm run dev

---

🌍 Environment Variables

Create a ".env.local" file.

NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
